Install Dishcovery on Your Android Device

How to Install Dishcovery

When you load this page, you should see a prompt to install the Dishcovery app. If not, follow these steps:

  1. Ensure you're using the latest version of Chrome or Brave.
  2. Tap the three-dot menu in the top-right corner.
  3. Select Install Dishcovery and follow the prompts.

If you encounter any issues, refresh the page or check browser compatibility.
